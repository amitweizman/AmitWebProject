About the Project:

This is a Data Science project on behalf the 10th grade class of Alterman High Tel Aviv.

This project includes 2 main areas of knowledge:

  • Data Science

Data Science



As part of the project, I located databases on the network and intend to build around them a website that allows access to the databases, data analysis and presentation to users in a convenient way. The site will allow the user to explore the data in the chosen field and try to reach valuable insights and conclusions.

The Tech Stack of the project is:

  • Python Programming
  • Pandas Library
  • Flask Web Framework
